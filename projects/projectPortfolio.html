Project Portfolio

This projectpage is about this particulary project and how I wanted to create my portfolio as a project to add to my portfolio - a bit confusing I know, so let me break it down. I love to program and write code to learn new tools and ways to code is something I find very interesting and fun. When presented this portfolio project in one of our courses I thoght of two different ways to do it:

  1. By finding a template such as wix.com or adobe portfolio and focus on creating a beautiful and fast portfolio. Or...
  2. Code it myself from scratch and thereby make it a project that I could say I have created and designed all by myself. I also knew this was the more time consuming approach.
Due to my realtivly new but big interest in programming I decided to go with the second option. This ended up being a much bigger project than first anticipated and it ended up including learning a new javascript library (react) and a new framework (next.js). This was probebly the hardest but also the most fun and interesting. I learned a lot in this process that I can use in my everyday/uni/work life. I'm no expert programmer but I know my way around code and its limitations.

How do I learn a new library, framework and way to code?

My first thought after choosing to program it myself was "What have I done? Where do I start? Can I actually do this?" and to be honest I wasen't sure for a really long time. I'm very lucky to have several programmers in my life that were able to help me but I really wanted to do it all myself to make sure I understood it (this dosen't mean that I didn't get help - I did a lot but I made sure that I understood everything).
